# openi-hackathon
OpenAI Build Week Hackathon project: Codex Session Guard

## Codex Session Guard

Codex Session Guard helps developers safely switch Codex contexts by making local Codex state visible, generating clean handoff packages, and quarantining old sessions instead of deleting them.

## Current Build Direction

The first build focuses on local core logic: repository branches, Codex worktrees, local Codex state, safe cleanup recommendations, and audit reports. The Electron UI and account connectors will be added after the local scanner is reliable.
